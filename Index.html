Grid vs Flexbox

Grid sirve para organizar la estructura general de la página en filas y columnas.

  • Ideal para estructuras de dos dimensiones.
  • Permite definir áreas claras con grid-template-areas.
  • Funciona muy bien para el layout principal.

Diferencia entre em/rem/hv

Em: Unidad relativa al tamaño de fuente del elemento padre.

Rem: Unidad relativa al tamaño de fuente del elemento raíz (html).

Hv: Unidad relativa a la altura de la ventana gráfica.
